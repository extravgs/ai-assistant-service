**Multi-purpose AI Assistant: QA & Sentiment Analysis Service with Docker**
=====================
**Что изменилось?:**
=====================
**- Контейнеризация через Docker: Весь проект теперь запускается одной командой.**
_____________________
**-Разделение ответственности:** 

   -Backend: Обрабатывает тяжелые задачи нейросетей (Hugging Face Transformers).
   
   -Bot: Отвечает только за взаимодействие с пользователем через aiogram.

_____________________

**-Связь через API: Бот общается с сервером по протоколу HTTP (FastAPI + httpx), что позволяет в будущем легко добавить Web-версию или мобильное приложение.**


_____________________
**-Безопасность: Настроено использование переменных окружения (.env) для защиты токенов.**



Как запустить
=====================

1. Убедитесь, что у вас установлен Docker Desktop.

_____________________

2. Создайте файл .env в корневой папке и добавьте туда свой токен:

```BOT_TOKEN=8200707484:AAG_... (ваш токен от @BotFather)```

```API_URL=http://backend:8000```

_____________________

3. Просто выполните команду в терминале:


```docker-compose up --build```

После завершения сборки бот будет готов к работе!

_____________________

Структура проекта

```/backend — FastAPI сервер с логикой нейросетей.```

```/bot — Код Telegram-бота и обработка команд.```

```docker-compose.yml — Инструкции по сборке и связке контейнеров.```
